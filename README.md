# Dice Roller - Ulauncher Extension

A simple and efficient dice roller extension for [Ulauncher](https://ulauncher.io/). Quickly roll any number of dice with any number of faces directly from your launcher.

![Icon](images/icon.png)

## Features

- **Custom Dice Rolls:** Specify the number of dice and the number of faces.
- **Result Summary:** View individual roll results and the total sum.
- **Copy to Clipboard:** Press Enter to copy the total sum to your clipboard.
- **Fast and Lightweight:** Designed for quick access and minimal resource usage.

## Installation

1. Open Ulauncher Preferences
2. Go to the **Extensions** tab
3. Click **Add Extension**
4. Paste the URL of this repository: `https://github.com/thomas-ehardt/ulauncher-diceroller`
5. Click **Add**

## Usage

Trigger the extension using the default keyword `dr` (or your custom keyword) followed by the number of dice and the number of faces.

**Syntax:**
`dr <number_of_dice> <number_of_faces>`

**Examples:**
- `dr 1 20`: Rolls one 20-sided die (1d20).
- `dr 2 6`: Rolls two 6-sided dice (2d6).
- `dr 3 10`: Rolls three 10-sided dice (3d10).

## Configuration

The trigger keyword can be customized in the Ulauncher extension settings (default is `dr`).
